# Student Management System

## 📌 Overview

The **Student Management System** is a Java console-based application developed as part of the **Syntecxhub Java Programming Internship (Week 2)**. It allows users to efficiently manage student records using a menu-driven interface.

## ✨ Features

* ➕ Add Student
* 📋 View Students
* 🔍 Search Student
* ✏️ Update Student Details
* 🗑️ Delete Student
* 🚪 Exit Application

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* VS Code
* Git & GitHub

## ▶️ How to Run

1. Compile the project:

```bash
javac *.java
```

2. Run the project:

```bash
java Main
```
